Welcome to SenaL



SenaL is machine learning end to end flask web app for "Sentiment Analysis" model created using Scikit-learn & VADER Sentiment.

This application will help you in the sentiment analysis of your given statament/article.

This is very user friendly and only takes your input and in one click it give you the value.
